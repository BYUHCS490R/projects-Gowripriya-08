What is normal flow in HTML?

In HTML, normal flow means the natural way a web page arranges elements before you add any special CSS positioning. By default, block elements (like div p. h1) stack on top of each other from top to bottom, each starting on a new line. Inline elements (like span a strong) stay in the same line and only take up as much space as their content. This basic, natural arrangement of elements is called the normal flow of a web page

What do absolute, fixed, and relative do?

Absolute: Positions an element relative to its nearest positioned ancestor. Fixed: Stays in the same place on the screen even when scrolling. Relative: Moves an element relative to its normal position.
