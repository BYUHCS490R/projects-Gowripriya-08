DEFINITIONS TO COMPUTER WORDS.

Internet:

The Internet is a big network that connects different computer devices around the world so they can share information and communicate eachother.

Web:

The web is a collection of webistes and pages that we can visit using web browser, like google browser, chrome and others.

Difeference between Internet and web:

Internet is like physical network that connects the computer whereas web is a webisite where you can see on your screen or it is a page. like think road is internet and car is webiste which drives on the road.

ASCII and Unicode:

ASCII is a older system that uses numbers to represent basic english letters,numbers and symbols Unicode is a new system that can show almost every character, including pther languages and emojis.

UTF-8:

UTF is a way of writing unicode character. it saves spaces by using fewer number for simple letters and more numbers for complex character.

URL:

URL means uniform Resource locator it is a web page. it tells your browser where to go to find a website

Domain Name:

A domain name is the essay to read name of a website like youtube.com . its simpler to remember than a long number code

HTTP:

HTTP means hyper text transfer protocol is a set of rules that help us to browser and a website talk to each other. It is not secure, so others could see the data being sent.

HTTPS:

IT means hyper text transfer protocol secure its a safe version of http. it protects information like passwords, bank details by encrypting it.

IP ADDRESS:

IP Address is like a home address for our computer on the internet. it tells others computer where to find it.

DNS:

DNS means domain name system it works like a phone book, it changes website names, into the number address for computer use.

W3C Recommendations:

It makes rules and guidelines so website can work well and look the same on all devices and browsers.

XML, XHTML, HTML, HTML5

XML: Used to store and move data in a neat way.

XHTML: A cleaner, stricter version of HTML.

HTML: The main language for making web pages.
