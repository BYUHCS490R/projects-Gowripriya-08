Why is the viewport meta tag important?

The viewport meta tag is important because it controls how a webpage looks on different devices, like phones, tablets, and computers. It helps make the website responsive, meaning it adjusts to fit the screen size so users don’t have to zoom in or scroll sideways.

2. How can you adjust the CSS for printing?

You can adjust the CSS for printing by using a printcss file by keeping nav as none. This lets you change how the page looks when someone prints it for example, hiding menus, removing background colors, or changing the font to make it easier to read on paper.

When would you want to adjust the CSS for printing?

we can adjust the CSS for printing when we want your web page to look clean and professional on paper. This is useful for pages like articles, receipts, reports, or forms that people might print out. It helps save ink and makes the printed version easier to read.
