When do you use a table?

Use a table when you need to show data that’s organized in rows and columns like schedules, prices, or comparisons.

When do you not use a table?

Don’t use a table just to arrange or format text or images on a page,use CSS for layout instead.

When do you need to do extra work for table accessibility?

Do extra accessibility work when our table has complex headers or spans multiple rows/columns—add proper header tags and descriptions so screen readers can understand it.
